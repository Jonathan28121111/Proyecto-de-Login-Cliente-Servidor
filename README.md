Este proyecto escolar está desarrollado en Java y utiliza la arquitectura cliente-servidor. Permite que varios clientes se conecten al servidor, 
realicen un proceso de inicio de sesión (login) y reciban mensajes desde el servidor. Además, el servidor incluye el juego "Adivina el número", 
donde los clientes pueden participar enviando sus intentos y recibiendo retroalimentación en tiempo real.

El código fuente incluye las clases principales para la comunicación entre cliente y servidor, manejo de usuarios y lógica del juego. 
El archivo `usuarios.txt` almacena los usuarios registrados.

Rama principal: `master`
